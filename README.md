# strapi-hook-allegro

This hook allows you to use [Allegro Rest API](https://developer.allegro.pl/documentation/) with [Strapi](https://github.com/strapi/strapi).

## Installation

```bash
# using yarn
yarn add strapi-hook-allegro

# using npm
npm install strapi-hook-allegro --save
```
